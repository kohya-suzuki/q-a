# README
時代に合った知識は、必ずしも学校が教えてくれるものではありません。
Samboは、家事や育児を担ってきた世代から次の世代へ、知識のバトンを渡すQ&Aアプリです。
これはみなさんにとっての最適なパートナー選びを強力にサポートします。
あなたが問題を作れば、どこかの誰かが解き、知恵の輪が一層広がっていきます。
ハイブリッドな次世代へ、プレゼントを残しませんか？
